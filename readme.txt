== Description ==

With WP Meta Tags you can add meta tags in posts, pages, categories and more...

= Features =

Meta Title Optimization
Meta Description Optimization
Meta Keywords Optimization
Index: Follow or Nofollow

WP Meta Tags is a simple but effective seo plugin.

Simply after plugin activation you will be redirected to the settings page of the plugin for configuration.

If you need help or you want to report a bug please visit http://seo.uk.net/wp-meta-tags/

== Installation ==

1. Upload the zip to 'plugins' directory or from WP Admin -> Plugins -> and search for Link to us.
1. Activate the plugin
1. You will be redirected to the settings page for configuration of the plugin.

If you need help with the plugin or you want to report a bug please refer to http://seo.uk.net/wp-meta-tags/
